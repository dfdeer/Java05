==================================================
1. 주제
==================================================

Java를 활용한 콘솔 기반 은행 입출금 시스템 구현

사용자가 계좌를 생성하고 로그인 후
입금, 출금, 송금, 잔액 조회 등의 기능을
사용할 수 있는 프로그램을 개발한다.

또한 Java의 객체지향 프로그래밍 구조와
메소드 활용, 예외 처리 등을 학습하는 것을 목표로 한다.
